# ReADOFAIMacro

使用 C++20 重写的 A Dance of Fire and Ice 宏工具项目

可以加载，解析并自动播放谱面

目前键盘输入部分使用Windows API，暂不支持DLC

目前没法很好地解析低速三连音、变速曲、拇指四指轮

## 使用:

在Main.cpp中修改keySequence来决定使用的按键

PlayScript()的第二个参数是最大硬抗BPM，也就是你希望宏在双指下最高能使用硬抗手法的BPM，高于这个BPM会用轮指手法，其他按键数的情况下宏会根据这个BPM自行调整，建议值在800~1100之间，越低手法越偏向轮指，越高手法越偏向硬抗

script.play()的第二个参数是一个按键，运行程序后在ADOFAI里打开谱面，在第一个轨道上点击这个按键就会开始播放谱面，建议使用ADOFAI Tweaks限制按键并选择一个不会影响打谱的按键(因为宏会把第一个轨道也算进手法)

构建程序后，运行需要先输入关卡文件路径(例: E:/adofai/level.adofai)

输入后点击前面指定的按键就可以播放了

播放期间可以把鼠标挪到屏幕的左上角(0,0)强制停止播放

## 构建:
`cmake --build build`
